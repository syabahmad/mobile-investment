Place your official app icon and splash images here to be used by Expo.

Recommended filenames (used by app.json):
- smartinvest-icon.png (1024x1024 PNG, square)
- smartinvest-splash.png (1242x2436 or 1242x2688 PNG)

I added lightweight SVG placeholders (`smartinvest-icon.svg`, `smartinvest-splash.svg`) so the app can build immediately; replace them with high-quality PNGs when available.

After adding the PNGs, rebuild or run the Expo dev server:

```bash
cd frontend
expo start
# or with EAS builds
eas build --platform android --profile preview
```
